Theory Block Library

The canonical vocabulary for the In-Depth Theory section, styled to the Locked Tokens. Every page is composed from these nine blocks — two of them interactive (mistake-check, compare-reveal). Each block renders below from its CONFIG object, with no hardcoded content, and the schema documented beside it. Live on the exemplar: Giant Covalent Structures →

Rule 1 — Colour = meaning (hard).
Semantic colour only where it encodes real meaning: green = conducts / mobile charge / favourable, red = does not / limitation. A plain fact or property is neutral. Decorative colour teaches a false pattern — never tint for looks.
Rule 2 — Pacing (hard).
One idea per block, short lines, distinct facts on their own line. If a block runs past ~2 lines of prose, it is two blocks.
Rule 3 — Parallel & mistake-first.
Compare-cards keep the same rows both sides so the eye scans across; mark the payoff row with highlightRow. Misconceptions are stated first, then corrected.
