Relative Atomic Mass, Atomic Number and Isotopes

Spec 5.1.1.6 📗 Foundation
📖 In-Depth Theory

Atomic Number and Mass Number

Every element is defined by its ATOMIC NUMBER (proton number):
ATOMIC NUMBER (Z) = number of PROTONS in the nucleus.
This is unique to each element — all carbon atoms have 6 protons, all iron atoms have 26.
In a neutral atom: protons = electrons.
MASS NUMBER (A) = total number of PROTONS + NEUTRONS in the nucleus.
From these two numbers:
Neutrons = mass number − atomic number
Example — sodium-23:
Mass number = 23, atomic number = 11.
Protons = 11, electrons = 11 (neutral), neutrons = 23 − 11 = 12.
In the periodic table, the ATOMIC NUMBER is the SMALLER number (it cannot exceed the mass number as you cannot have negative neutrons).

Isotopes

ISOTOPES are atoms of the SAME ELEMENT with the SAME ATOMIC NUMBER but DIFFERENT MASS NUMBERS.
Same protons → same element → same chemical properties.
Different neutrons → different mass → different physical properties (e.g. density, rate of diffusion).
Isotopes have IDENTICAL CHEMICAL behaviour — because chemical reactions depend on electron configuration (determined by proton number, which is the same).
Examples:
CARBON ISOTOPES:
Carbon-12 (¹²C): 6p + 6n — the standard (98.9% of carbon)
Carbon-13 (¹³C): 6p + 7n — stable, ~1.1%
Carbon-14 (¹⁴C): 6p + 8n — radioactive, used in carbon dating
CHLORINE ISOTOPES:
Chlorine-35 (³⁵Cl): 17p + 18n — ~75% of chlorine
Chlorine-37 (³⁷Cl): 17p + 20n — ~25% of chlorine
Because of this mixture, the Ar of chlorine ≈ 35.5 — between the two isotope masses.

Relative Atomic Mass

The RELATIVE ATOMIC MASS (Ar) is the WEIGHTED AVERAGE mass of all atoms of an element compared to 1/12 of the mass of a carbon-12 atom.
Because most elements have multiple isotopes in different abundances, Ar is NOT a whole number.
Formula:
Ar = Σ (% abundance × mass number) ÷ 100
Example — chlorine:
Ar = (75 × 35 + 25 × 37) ÷ 100
Ar = (2625 + 925) ÷ 100
Ar = 3550 ÷ 100 = 35.5
This explains why periodic table Ar values are often not whole numbers — they are weighted averages across all naturally occurring isotopes.
⚠️ Common Mistake

Ar is a WEIGHTED average — not a simple average. Chlorine has Ar 35.5 because ~75% is Cl-35 and only ~25% is Cl-37. If both were 50/50, Ar would be 36. The heavier isotope has LESS influence because it's less abundant. Also: isotopes have SAME chemical properties (same electrons) but DIFFERENT physical properties (different mass).

📐 Key Equations
Mass number = protons + neutrons
Neutrons = mass number − atomic number
Ar = Σ(% abundance × mass number) ÷ 100
📌 Key Note

Atomic number = protons. Mass number = protons + neutrons. Neutrons = mass number − atomic number. Isotopes: same element, same protons, different neutrons — same chemical, different physical properties. Ar = weighted average of isotope masses.
