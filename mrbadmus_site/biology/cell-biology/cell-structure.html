⚡ Animal, Plant and Prokaryotic Cells

Spec 4.1.1 Paper 1
📖 In-Depth Theory

Animal Cells — The Basics

Nucleus: contains DNA — controls cell activity.

Cell membrane: controls what enters and leaves.

Cytoplasm: where most chemical reactions happen.

Mitochondria: site of aerobic respiration — produces ATP energy.

Ribosomes: where proteins are made.

Plant Cells — Extra Components

All animal cell components PLUS:

Cell wall: made of cellulose — provides support and shape.

Chloroplasts: contain chlorophyll — absorb light for photosynthesis.

Permanent vacuole: filled with cell sap — helps maintain cell shape (turgor pressure).

Some plant cells lack chloroplasts (e.g. root cells — no light underground).

Prokaryotic Cells (Bacteria)

Much smaller than eukaryotic cells.

NO nucleus — DNA is a single loop in the cytoplasm.

NO membrane-bound organelles.

Have: cell membrane, cell wall (not cellulose), cytoplasm, ribosomes.

May have: plasmids (small circular DNA loops), flagella (tail for movement), capsule (slime layer).

Microscopy

Light microscopes: use light — max magnification ~×2000, can see cells and large organelles.

Electron microscopes: use electrons — max magnification ~×2,000,000, can see mitochondria, ribosomes in detail.

Magnification = image size ÷ actual size.

Resolution: ability to distinguish two close points as separate — electron microscopes have much higher resolution.

⚠️ Common Mistake

Plant cells don't ALL have chloroplasts — only cells that are exposed to light. Root cells and storage cells don't have them. Also — prokaryotic cells have NO nucleus and NO membrane-bound organelles.

📐 Variables
MMagnification (M) is measured in no unit (×)
IImage size (I) is measured in millimetres or micrometres (mm / µm)
AActual size (A) is measured in millimetres or micrometres (mm / µm)
📐 Key Equations
Magnification = image size ÷ actual size
📌 Key Note

1 mm = 1000 µm. 1 µm = 1000 nm. Scale bars on micrographs are your friend — use them to calculate actual size.
