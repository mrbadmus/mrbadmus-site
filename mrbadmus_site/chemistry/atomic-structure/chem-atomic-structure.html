⚡ Atomic Structure and the Periodic Table

Spec 4.1.1 Paper 1
📖 In-Depth Theory

Inside Every Atom

Every atom has a nucleus containing protons (+) and neutrons (no charge).

Electrons (−) orbit the nucleus in shells.

In a neutral atom: number of protons = number of electrons.

The atom is mostly empty space — the nucleus is tiny compared to the whole atom.

The Three Subatomic Particles

Proton: relative mass = 1, relative charge = +1, in nucleus.

Neutron: relative mass = 1, relative charge = 0, in nucleus.

Electron: relative mass ≈ 0, relative charge = −1, in shells.

Atomic number (Z) = number of protons.

Mass number (A) = protons + neutrons.

Neutrons = A − Z.

Electronic Structure

Electrons fill shells from inside out.

Shell 1: maximum 2 electrons.

Shell 2: maximum 8 electrons.

Shell 3: maximum 8 electrons (GCSE level).

Example: Sodium (Z=11) → 2, 8, 1.

Outer shell electrons determine chemical properties.

The Periodic Table

Elements arranged in order of atomic number.

Period (row) = number of electron shells.

Group (column) = number of outer shell electrons.

Group 1: 1 outer electron — very reactive metals.

Group 7: 7 outer electrons — reactive non-metals.

Group 0: full outer shell — very unreactive noble gases.

⚠️ Common Mistake

Mass number = protons + neutrons (not protons + electrons — electrons have negligible mass). Also: Group 0 noble gases have a FULL outer shell, not zero electrons.

📐 Key Equations
Neutrons = mass number − atomic number
📌 Key Note

Period = number of shells. Group = number of outer electrons.
